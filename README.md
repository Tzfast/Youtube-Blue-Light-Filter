<h1>How to Install It</h1>

**1-** Copy Blue Light Filter.js script
<p></p>

**2-** Install Tampermonkey Extension if you still don't have it
<p></p>

**3-** Open Tampermonkey menu and click on "Create A New Script"

<img src="https://i.imgur.com/29iKb07.png">
<p></p>

**4-** Delete everything at the editor, paste the script there and then save it
<p></p>

**5-** Enable the script if it's not enabled

<img src="https://i.imgur.com/ZkfC3dQ.png">
<p></p>

**6-** Have fun!
